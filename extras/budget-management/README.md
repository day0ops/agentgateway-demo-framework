# Budget Management

For small tests with gpt-4o-mini, the cost is much lower:

GPT-4o-mini Cost

┌────────┬────────────────────┬────────────────────────────────┐
│ │ Per Million Tokens │ Per Request (~50 in, ~300 out) │
├────────┼────────────────────┼────────────────────────────────┤
│ Input │ $0.15 │ $0.0000075 │
├────────┼────────────────────┼────────────────────────────────┤
│ Output │ $0.60 │ $0.00018 │
├────────┼────────────────────┼────────────────────────────────┤
│ Total │ │ ~$0.0002 │
└────────┴────────────────────┴────────────────────────────────┘

That's about 25x cheaper than Claude Sonnet!

Recommended Budgets for GPT-4o-mini

To see warning after 1-2 requests:

Org Budget: $0.0006 (warning at $0.00048 → ~2-3 requests)
Team Budget: $0.0003 (warning at $0.00024 → ~1 request)

Or slightly larger for a few more requests:

Org Budget: $0.001 (warning at $0.0008 → ~4 requests)
Team Budget: $0.0005 (warning at $0.0004 → ~2 requests)
